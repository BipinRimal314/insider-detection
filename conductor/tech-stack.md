# Technology Stack

## Core Language
- **Python**: Primary programming language for the entire pipeline.

## Data Processing & Engineering
- **Polars**: High-performance DataFrame library used for efficient data loading and preprocessing.
- **Pandas**: Used for data manipulation and compatibility with other libraries.
- **NumPy**: Fundamental package for scientific computing and array manipulation.

## Machine Learning & Deep Learning
- **TensorFlow / Keras**: Deep learning framework used for building and training the LSTM Autoencoder and Deep Clustering models.
- **Scikit-learn**: Machine learning library used for the Isolation Forest model and evaluation metrics.

## Visualization
- **Matplotlib**: Core plotting library for generating static visualizations.
- **Seaborn**: Statistical data visualization library based on Matplotlib, used for high-level plotting.
